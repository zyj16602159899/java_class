1.java语言介绍
    1.java是一门后台开发语言
    2.大型web系统开发，APP后台服务器首先开发语言
    3.特点：跨平台、安全性高、有丰富的类库
    4.核心思想：面向对象
2.JDK（开发工具）
    为什么要安装jdk？
        1.jdk是java development kit（java开发工具）
        2.jdk提供了丰富的命令，比如编译、执行
        3.jdk提供了jre(java运行环境)和丰富的类库，可以帮助我们完成程序开发
    为什么要配置环境变量？
        配置了环境变量可以保证我们在系统的任意目录调用到java命令（java、javac）
