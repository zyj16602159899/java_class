一、JavaScript的使用方式：
1.行内式（主要用于事件）
2.内嵌式
3.外链式

二、变量和数据类型
1.定义变量
JS是一种弱类型语言，即不需要指定变量的类型，变量的类型由它的值来决定。定义变量需要使用关联字var，并以;结尾
定义变量的语法格式：var 变量名 = 值;
同时定义多个变量，可用“,”隔开，共用一个var

2.JS的注释
单行注释：// 注释内容
多行注释：/* 注释内容 */

3.数据类型
JS有六种数据类型，包括五种基本数据类型和一种复杂数据类型（object）
number:数字类型
string:字符串类型
boolean:布尔类型，true或false
undefined:变量声明未初始化，它的值就是undefined
null:表示空对象，如果定义的变量将来准备保存对象，可以将变量初始化为null，在页面获取不到对象，返回的值就是null
object:JS对象属于复合类型

4.变量命名规范
（1）区分大小写
（2）第一个字符必须是字母、下划线、或者美元符号（$）
（3）其他字符可以是字母、下划线、美元符号或数字

三、函数定义和调用
1.函数定义
函数就是可以重复使用的代码块，用关键字function来定义函数
2.函数调用
（1）函数调用就是函数名+小括号，比如：函数名(参数)
（2）定义有参数有返回值的函数：
定义函数时，如果有参数，参数放到小括号里面；如果有返回值，返回值通过return关键字来返回

四、变量作用域
1.变量作用域就是变量的使用范围，变量分为：局部变量、全局变量
2.局部变量就是在函数内定义的变量，只能在函数内部使用
3.全局变量就是在函数外定义的变量，可以在不同的函数中使用，并可共享全局变量

五、条件语句
1.语法：
if语句；
if..else..语句；
if..else..if..else..语句

2.比较运算符：
==  等于（只对比值，不管类型）
=== 全等于（值和类型）
!=  不等于
>   大于
<   小于
>=  大于或等于
<=  小于或等于
