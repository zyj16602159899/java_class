二、变量和数据类型
1.定义变量
JS是一种弱类型语言，即不需要指定变量的类型，变量的类型由它的值来决定。定义变量需要使用关联字var，并以;结尾
定义变量的语法格式：var 变量名 = 值;
同时定义多个变量，可用“,”隔开，共用一个var

2.JS的注释
单行注释：// 注释内容
多行注释：/* 注释内容 */

3.数据类型
JS有六种数据类型，包括五种基本数据类型和一种复杂数据类型（object）
number:数字类型
string:字符串类型
boolean:布尔类型，true或false
undefined:变量声明未初始化，它的值就是undefined
null:表示空对象，如果定义的变量将来准备保存对象，可以将变量初始化为null，在页面获取不到对象，返回的值就是null
object:JS对象属于复合类型

4.变量命名规范
（1）区分大小写
（2）第一个字符必须是字母、下划线、或者美元符号（$）
（3）其他字符可以是字母、下划线、美元符号或数字

三、函数定义和调用
1.函数定义
函数就是可以重复使用的代码块，用关键字function来定义函数
2.函数调用
（1）函数调用就是函数名+小括号，比如：函数名(参数)
（2）定义有参数有返回值的函数：
        定义函数时，如果有参数，参数放到小括号里面；如果有返回值，返回值通过return关键字来返回

四、变量作用域
1.变量作用域就是变量的使用范围，变量分为：局部变量、全局变量
2.局部变量就是在函数内定义的变量，只能在函数内部使用
3.全局变量就是在函数外定义的变量，可以在不同的函数中使用，并可共享全局变量

六、数组
1.数组就是一组数据的集合，JS中数组里面的数据可以是不同类型的数据，好比python中列表。
2.数组的定义：
    （1）实例化对象方式创建
    var aList = new Array(1,2,3);
    （2）字面量方式创建，推荐使用
    var aList2 = [1,2,3,"aaa"];
3.多维数组：指的是数组的成员也是数组，把这样的数组叫做多维数组
var aList = [[1,2,3],["a","b"]];
4.数组的操作
    （1）获取数组的长度
        alert(aList.length);
    （2）根据下标取值
        alert(aList[0]);
    （3）从数组最后添加和删除数据
        添加：aList.push(5);
        删除：aList.pop();
    （4）根据下标添加和删除数据
        aList.splice(start,num,element1,...,elementN)
        解析：
            start:必需，开始删除的索引
            num:可选，删除数组元素的个数
            elementN:可选，在start索引位置要插入的新元素
        此方法会删除从start索引开始的num个元素，并将elementN参数插入到start索引位置

九、定时器
1.定时器就是在一段特定的时间后执行某段程序的代码。
2.定时器的使用：
    （1）setTimeout(func,delay,param1,param2,...)：以指定的时间间隔（毫秒）调用一次函数的定时器
    （2）setInterval(func,delay,param1,param2,...)：以指定的时间间隔（毫秒）重复调用一个函数的定时器
    参数说明：
    func:表示定时器要执行的函数名
    delay:表示时间间隔，默认是0，单位是毫秒
    param1:表示定时器执行函数的第一个参数，以此类推传入多个执行函数（func）对应的参数
